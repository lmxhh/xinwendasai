# 数说十年中原粮仓：方寸耕地里的产能蜕变

## 项目简介

本项目是一个数据新闻可视化网页，通过图表和地图展示河南省2016-2025年十年间粮食产能的变化与发展。

## 技术栈

- HTML5 + CSS3
- JavaScript (ES6+)
- ECharts 5.5.0
- GSAP 3.12.5
- CountUp.js 2.8.0

## 功能特性

- 📊 数据可视化：多种图表展示粮食产量、播种面积、单产变化等数据
- 🗺️ 交互式地图：河南地图SVG轮廓，支持悬停高亮和点击跳转
- 📈 动态图表：各地市十年产能变化轮播图
- ✨ 动画效果：滚动触发动画、数字计数动画
- 📱 响应式设计：适配不同屏幕尺寸
